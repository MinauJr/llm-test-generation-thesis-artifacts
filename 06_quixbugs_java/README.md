# QuixBugs Java artefacts

This directory contains cleaned artefacts for the QuixBugs Java experiments.

Structure:

- `suts/`: prepared QuixBugs Java SUTs only.
- `suts_patched_evosuite/`: patched EvoSuite SUT material, when present.
- `scripts/`: analysis and selected generation/configuration scripts.
- `results/analysis_tables/`: final CSV tables used in the dissertation.
- `results/cluster_summaries/`: compact summaries for the open-weight cluster execution.
- `results/gpt55_audit/`: selected GPT-5.5 audit/closure files.

Raw generated tests, repeated model outputs, logs, figures, backups, Python SUTs and build artefacts were intentionally excluded.
