# QuixBugs Python artefacts

This directory contains the cleaned artefacts for the QuixBugs Python dataset.

Structure:

- `suts/`: prepared Python SUTs used in the experiments.
- `scripts/`: generation/evaluation/analysis scripts and prompts related to QuixBugs Python.
- `results/`: cleaned result summaries and metadata from the final runs.

Large raw logs, caches, generated temporary work directories, build artefacts and intermediate outputs were intentionally excluded.
