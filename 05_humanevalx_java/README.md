# HumanEval-X Java artefacts

This directory contains cleaned artefacts for the HumanEval-X Java experiments.

Structure:

- `suts/`: prepared HumanEval-X Java SUTs.
- `suts_patched_evosuite/`: patched SUT material used for EvoSuite-related execution.
- `scripts/`: analysis scripts used to rebuild the final tables/figures.
- `results/analysis_tables/`: final CSV tables used in the dissertation.
- `results/cluster_metadata/`: compact metadata for the open-weight cluster execution.

Raw cluster generations, repeated model outputs, PNG figures, logs, backups and build artefacts were intentionally excluded.
